# Wondershaper QuickToggle v1.1.1 released

Good news! v1.1.1 is out with a small but important fix for the tray tooltip.

## What changed
- Fix tray icon tooltip showing "Main.py" instead of the app name.
- Add `setproctitle` to set the process name correctly.

## Download
- Release page: https://github.com/Danux-Be/Wondershaper-GUI/releases/tag/v1.1.1

## Quick install
```bash
wget https://github.com/Danux-Be/Wondershaper-GUI/releases/download/v1.1.1/wondershaper-quicktoggle_1.1.1-1_all.deb
sudo dpkg -i wondershaper-quicktoggle_1.1.1-1_all.deb
```
